Installation

NPM 
$ npm install --save @ngx-gallery/core @ngx-gallery/lightbox @ngx-gallery/gallerize @angular/cdk
YARN 
$ yarn add @ngx-gallery/core @ngx-gallery/lightbox @ngx-gallery/gallerize @angular/cdk

Usage

1

Import GalleryModule in the root module

2

Import gallery styles src/styles.scss


@import '~@ngx-gallery/core/styles/gallery';
@import '~@ngx-gallery/core/styles/lightbox';

3

Add [gallerize] on images container

See Auto-detect Example
