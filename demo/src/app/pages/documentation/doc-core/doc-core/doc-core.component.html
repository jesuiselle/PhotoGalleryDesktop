Installation

NPM 
$ npm install --save @ngx-gallery/core
YARN 
$ yarn add @ngx-gallery/core

Usage

1

Import GalleryModule in the root module

2

Import gallery styles src/styles.scss


@import '~@ngx-gallery/core/styles/gallery';

3

Load images into the gallery

Gesture Support (optional)

@ngx-gallery relies on HammerJS for gestures, make sure it is loaded into the application.

You can add HammerJS to your application via npm, a CDN (such as the Google CDN), or served directly from your app.

NPM 
    $ npm install --save hammerjs
YARN 
    $ yarn add hammerjs

After installing, import HammerJS into src/polyfills.ts

    import 'hammerjs';
